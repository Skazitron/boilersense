CGT 24800 - Visual Effects - Pyrotechnics And Destruction Effects

Credit Hours: 3.00.  This course focuses on the application of visual effects methodology to create effects based on destruction techniques and pyrotechnics. Students implement projects using methods for simulating destruction, fire, and smoke. Emphasis is placed on using production tools and techniques to produce realistic and aesthetically pleasing graphics.

Learning Outcomes: 
1.  Implement destruction simulations using production tools and techniques.
2.  Implement fire and smoke simulations using production tools and techniques.
3.  Integrate destruction and fire simulations to meet the needs of 3rd party video compositing team.
4.  Render simulations realistically within the provided time and computational constraints.

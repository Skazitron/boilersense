CNIT 28000 - Systems Analysis And Design Methods

Credit Hours: 3.00.  Comprehensive introduction to information systems development. Topics include the systems analyst, the systems development life cycle, methodologies, development technology, systems planning, project management, systems analysis, systems design, systems implementation, and systems support. Introduction to tools and techniques for systems development.

Learning Outcomes: 
1.  Identify the stages of the systems development life cycle and the different systems methodologies used to implement it.
2.  Apply project management techniques and tools for systems development.
3.  Apply requirements analysis strategies and requirements gathering techniques to determine business requirements.
4.  Analyze an as-is system or a to-be system by applying appropriate tools for analysis modeling.
5.  Design a system by applying appropriate tools for design modeling.

CGT 31500 - Computer Graphics Programming II

Credit Hours: 3.00.  A continuation of the study of computer graphics programming concepts with an emphasis on interactive graphics and application development.  Topics include polygon tessellation, vector imaging, interactive programming techniques, application development, and game development.

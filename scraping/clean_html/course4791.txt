CNIT 30600 - Game Development I: Core Skills And Technologies

Credit Hours: 3.00.  (CGT 24500) This course introduces students to the fundamental technologies and skill sets required to develop video games. Students will be taught agile development methodologies in a team-based environment.

Learning Outcomes: 
1.  Demonstrate a moderate level of individual skill with a commercial game engine.
2.  Demonstrate the ability to observe game mechanics and reproduce them.
3.  Acquire competency in each of the following domains: 3D and 2D level creation, basic art asset creation, visual programming basics, implementing game mechanics.
4.  Demonstrate the ability to complete a market-ready video game in a team-based production environment.
5.  Demonstrate the ability to implement agile production techniques.

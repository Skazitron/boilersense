MFET 36300 - Intelligent Manufacturing Systems I

Credit Hours: 3.00.  This course investigates how artificial intelligence, industrial IoT, and big data are used to enable predictive analytics with machine learning. Emphasis is on optimizing production processes, improving productivity, quality, efficiency, and integrating predictive maintenance applications in smart manufacturing operations. Typically offered Fall Spring Summer.

Learning Outcomes: 
1.  Apply appropriate machine learning algorithms for improving and optimizing manufacturing operations.
2.  Enable predictive analytics with machine learning for maintenance operations.
3.  Evaluate manufacturing processes using machine learning.
4.  Identify and solve manufacturing problems using machine learning and root cause analysis.
5.  Describe how data- and knowledge-based systems are used in product and design lifecycles.

CGT 62000 - Graphics Processing Unit Computing

Credit Hours: 3.00.  This course provides a working knowledge of general-purpose graphics processing unit computing (GPGPU) using CUDA, OpenCL,  and C++. OpenCL and CUDA are the platform independent industrial standard API and leading edge technology for GPGPU. The course focuses on utilizing, GPU for advanced tasks of scientific computing on parallelization of creating real-time and interactive applications and is structured into several blocks; parallel programming and synchronization tasks, GPU architecture, CUDA programming model, the OpenCL programming model.

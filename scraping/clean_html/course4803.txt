CNIT 34210 - Storage Area Networking

Credit Hours: 2.00.  This course introduces storage area network (SAN) technologies. Storage area networking concepts, architectures, protocols, and best practice implementation techniques are examined. Techniques for remotely booting systems over a SAN are covered. Students implement and maintain an industry standard SAN and network booting solutions.

Learning Outcomes: 
1.  Compare and contrast various enterprise storage architectures.
2.  Implement and maintain storage area network services.
3.  Utilize storage area network services from different client operating systems.
4.  Install and configure a network boot environment for multiple operating systems.
